# Laboratornaya-3 
Для компиляции используется CMakeLists.txt
Порядок компиляции
1. mkdir build
2. cd build
3. cmake ..
4. cmake --build .
5. calculator.exe
6. вводите то, что хотите от программы
Режимы:
1) evaluate [количество переменных] [имена переменных] [значения переменных] [выражение] - вычисляет значение выражения. Пример ввода: evaluate 2 x y 3 4 x*x+y*2
2) derivative [количество переменных] [имена переменных] [значения переменных не важны] [выражение] - вычисляет производную от выражения (Здесь значения переменных не важны, но их нужно указать (можно поставить 0)). Пример ввода: derivative 1 x 0 x^2
3) evaluate_derivative [количество переменных] [имена переменных] [значения переменных] [выражение] - вычисляет значение производной в точке. Пример ввода: evaluate_derivative 1 x 2 x^3
4) [любое выражение без ключа] - проверяет как работает лексер (нет ключевого слова!)
